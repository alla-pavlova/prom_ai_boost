# Prom AI Boost

CLI-инструмент для обработки карточек товаров Prom.ua.

## Возможности

- Чтение Excel (.xlsx)
- Генерация описаний
- Генерация SEO-тегов
- HTML-описание товара
- Контроль дублей по SKU
- Логирование ошибок
- Статус обработки
- Дата обработки

## Установка

```bash
pip install -r requirements.txt
